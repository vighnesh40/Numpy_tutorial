# NumPy Tutorial 🧮

A beginner-friendly NumPy tutorial created in Jupyter Notebook that covers the fundamentals of NumPy for data analysis and scientific computing using Python.

## 📌 Project Overview

This notebook demonstrates the essential concepts of NumPy with practical examples. It is designed for students and beginners who want to learn how to perform numerical operations efficiently using Python.

## 📚 Topics Covered

- Creating NumPy Arrays
- Array Attributes
  - Shape
  - Size
  - Data Type
  - Dimensions
- Array Initialization Methods
  - Zeros
  - Ones
  - Full
  - Identity Matrix
  - Empty Arrays
- Creating Arrays with `arange()` and `linspace()`
- Random Number Generation
- Array Indexing
- Array Slicing
- Array Reshaping
- Flattening Arrays
- Array Stacking
- Array Splitting
- Mathematical Operations
- Operations on Multiple Arrays
- Statistical Functions
- Array Comparison
- Broadcasting
- Handling Null and Infinite Values
- Saving and Loading NumPy Arrays

## 🛠️ Technologies Used

- Python 3
- NumPy
- Jupyter Notebook

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/numpy-tutorial.git
```

### 2. Navigate to the project folder

```bash
cd numpy-tutorial
```

### 3. Install NumPy

```bash
pip install numpy
```

### 4. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open `numpy_tutorial.ipynb` and run the cells sequentially.

## 🎯 Learning Outcomes

After completing this notebook, you will be able to:

- Create and manipulate NumPy arrays
- Perform mathematical operations efficiently
- Understand array indexing and slicing
- Reshape and combine arrays
- Generate random data
- Apply statistical functions
- Handle missing and infinite values
